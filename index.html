Tic Tac Toe

Rules

The rules of the game are very simple.

  1. Tic Tac Toe (T.T.T) is played on a 3 square grid system.
  2. The first player is chooses a token (either X or O) and the second player is assigned the unselected token.
  3. Players will place their tokens on individuals cells inside the grid and they cannot repeat.
  4. The first player to get 3 of their tokens in a row (up, down, across, or diagonally) * ;) wins the game.
  5. If all 9 cells are full, the game is over. If no player has 3 marks in a row, the game ends in a tie.
